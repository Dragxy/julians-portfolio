Linsinger Vermessung

The Challenge

Modern businesses rely heavily on stable and secure network systems, and Linsinger Vermessung was no exception. However, their existing network setup, while functional, needed a detailed review to ensure scalability, efficiency, and documentation for future maintenance. My task was to:

  • Map out the network infrastructure: Identify and document all network components, including servers, LAN, WAN, switches, and other devices.
  • Evaluate the server rooms: Assess the organization, connectivity, and cooling systems in their server rooms.
  • Propose improvements: Recommend practical upgrades or organizational changes to enhance reliability and performance.

The Process

1. Network Mapping and Documentation

I began by systematically mapping the network. Using tools and manual checks, I documented:

  • Physical connections: cables, switches, and routers.
  • Logical connections: IP configurations, subnets, and access points.
  • Devices and services: Servers, workstations, and their respective roles within the network.

Creating a visual network map not only clarified the existing setup but also highlighted areas that could benefit from optimization.

2. Server Room Assessment

The server rooms were a critical focus. I evaluated:

  • Cable management: Identifying areas where tangled cables or unclear labeling could lead to inefficiencies.
  • Cooling systems: Checking airflow and ensuring that overheating wasn’t a potential issue.
  • Hardware organization: Reviewing rack layouts to identify opportunities for better utilization of space.

3. Proposing Improvements

Based on my findings, I provided a set of actionable recommendations:

  • Enhanced cable management: Suggested labeling systems and rerouting for clearer identification and troubleshooting.
  • Upgraded network devices: Highlighted outdated hardware that could be replaced for better performance.
  • Documentation updates: Delivered a comprehensive network manual, including diagrams and configuration details, for internal use.

Results

By the end of the week, I presented my findings and recommendations to the IT department. While some improvements, like better labeling, were implemented immediately, others were added to their long-term roadmap. The visual documentation I created became a reference tool for their ongoing network management, streamlining future maintenance and expansion efforts.

Takeaways

This project reinforced my skills in:

  • Network analysis: Mapping and documenting complex systems efficiently.
  • Problem-solving: Identifying and addressing gaps in infrastructure.
  • Communication: Translating technical findings into actionable insights for the team.

Though short, this experience at Linsinger Vermessung was incredibly rewarding. It allowed me to apply my knowledge in a real-world setting and contribute meaningfully to a company’s technical operations.

If your business is considering a network review or upgrade, remember—sometimes a fresh set of eyes can make all the difference!
